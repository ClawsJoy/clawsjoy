# 香港故事连续剧 - 项目记忆

## 剧集设定
- 主题：香港优才计划
- 主角：李浩（年轻人通过优才计划来港）
- 时长：3分钟/集
- 格式：旁白稿（不要剧本格式）

## 第一集摘要（假设）
《梦想启航》：李浩在广州申请，获批后来到香港，站在维港憧憬未来。
悬念：新的生活刚刚开始，挑战接踵而至。

## 第二集摘要（已制作）
《追梦启航》：李浩初到香港，在公司加班时意外发现一份重要文件遗失，
他决定自己寻找，过程中与关键人物产生交集。
悬念：文件的重要性超乎想象，李浩面临命运转折点。

## 第三集要求（新）
- 标题：《抉择时刻》
- 承接第二集悬念：李浩如何处理这份重要文件？
- 剧情发展：他必须决定是自己处理还是上报公司
- 高潮：文件内容揭示了一个关于公司项目的真相
- 悬念：李浩的选择将如何影响他的职业生涯？

## 风格偏好
- 叙事性强，情感丰富
- 开头要有画面感（李浩看着手中的文件，内心挣扎）
- 结尾留悬念（下集预告：真相浮出水面）

## 原子技能
- story_outline → 生成大纲
- script_writer → 旁白稿
- tts → 语音
- ffmpeg_video → 视频（支持字幕）
