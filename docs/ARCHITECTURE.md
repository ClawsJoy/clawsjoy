#核心流程
技能执行流程
用户输入 → API Gateway → SkillLoader → 技能执行 → 返回结果
                              ↓
                        记忆系统记录
#Agent 调度流程
请求 → 注册中心 → 能力匹配 → Agent 选择 → 执行 → 返回
#数据流
1.用户请求 → API Gateway

2.技能加载器查找技能

3.执行技能逻辑

4.结果存入记忆系统

5.返回给用户

#扩展性设计
#技能系统: 即插即用，自动发现

#Agent 系统: 注册中心管理，动态路由

#配置中心: 统一配置，环境变量覆盖
