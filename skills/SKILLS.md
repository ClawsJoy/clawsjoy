# ClawsJoy 原子技能清单

## 核心调度 (core/)
| 技能 | 功能 |
|------|------|
| do_anything | 大脑调度器，自主规划执行 |
| calibrated_executor | 校准执行器 |
| quality_gate | 质量门禁 |

## 数学计算 (math/)
| 技能 | 功能 |
|------|------|
| add | 加法 |
| multiply | 乘法 |
| divide | 除法 |
| calculator | 计算器 |

| 技能 | 功能 |
|------|------|
| remove_bg | 抠图，移除背景 |
| spider | 图片采集 |
| image_slideshow | 图片轮播 |
| character_designer | 角色设计 |
| ai_image | AI 图片生成 |

## 视频制作 (video/)
| 技能 | 功能 |
|------|------|
| manju_maker | 漫剧视频制作 |
| video_uploader | 视频上传 |
| add_subtitles | 添加字幕 |
| ffmpeg_video | FFmpeg 视频处理 |

## 数据处理 (data/)
