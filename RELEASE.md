# ClawsJoy 3.0 Release Notes

## 🎉 新特性

- **97+ 原子技能**：覆盖数学、图像、视频、文本、网络等领域
- **9 个智能 Agent**：代码、视频、YouTube、数据、内容、爬虫、测试、监控、通知
- **统一注册中心**：服务注册、技能注册、Agent 注册一体化
- **完整数据闭环**：记忆存储、智能分析、决策引擎、自适应调优
- **Web Dashboard**：可视化监控和管理
- **API 文档**：Swagger/OpenAPI 自动生成
- **Prometheus 监控**：性能指标采集

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/your-repo/clawsjoy.git
cd clawsjoy

# 安装
./install.sh

# 启动
./start_all.sh
