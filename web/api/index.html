📖 Swagger UI（交互式文档）

在线测试 API 接口，实时查看请求/响应

http://localhost:8094/api/docs →

🔗 核心 API 端点

认证服务 (8092)

POST/api/auth/login - 用户登录
GET/api/auth/health - 健康检查

租户管理 (8088)

GET/api/tenants - 列出租户
POST/api/tenants - 创建租户

计费系统 (8090)

GET/api/billing/balance?tenant_id=1 - 查询余额
POST/api/billing/recharge - 充值

咖啡订购 (8085)

GET/api/coffee/shops - 咖啡店列表
GET/api/coffee/menu?shop_id=1 - 菜单查询
POST/api/coffee/order - 下单

Workflow (8093)

GET/api/workflow/list - Workflow 列表
GET/api/workflow/detail?workflow_id=xxx - Workflow 详情

📦 快速测试命令

curl http://localhost:8088/api/tenants
